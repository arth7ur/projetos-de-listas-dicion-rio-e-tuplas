# Exercícios de Python - Dicionários e Listas

Este repositório contém exercícios práticos desenvolvidos durante o curso de Python, com foco em estruturas de dados compostas.

## 🚀 Desafios Incluídos:
* **dicionariotreinamento2.py**: Script que gerencia o cadastro de estados e siglas usando dicionários dentro de listas, com tratamento de cópia de dados (`.copy()`).

## 🛠️ Tecnologias e Conceitos:
* Python 3
* Manipulação de Dicionários (`dict`)
* Listas compostas
* Estruturas de repetição (`for`)
